# 贪吃蛇游戏

这是一个使用HTML, CSS和JavaScript开发的经典贪吃蛇游戏。

## 功能特点

- 使用键盘方向键控制蛇的移动
- 实时显示游戏得分
- 可调节游戏速度的滑块控制
- 现代化的游戏界面设计

## 如何运行

1. 克隆或下载此仓库到本地
2. 使用浏览器打开`index.html`文件
3. 或者使用Python的HTTP服务器运行:
   ```
   python -m http.server 8000
   ```
   然后在浏览器中访问 `http://localhost:8000`

## 游戏控制

- **方向键**: 控制蛇的移动方向
- **速度滑块**: 调整蛇的移动速度

## 技术栈

- HTML5 Canvas: 用于游戏渲染
- CSS3: 用于样式和布局
- JavaScript: 用于游戏逻辑和交互

## 文件结构

- `index.html`: 游戏的HTML结构
- `style.css`: 游戏的样式表
- `script.js`: 游戏的JavaScript逻辑
